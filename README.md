# Installation

Öncelikle projeyi clonlayalım.

> git clone https://github.com/mFatihGorhan/kodluyoruzilkrepo.git

Bu kodu terminale yazıp repomuzu indiriyoruz.

> cd kodluyoruzilkrepo
> code .
